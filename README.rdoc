= redmine_improved_searchbox

Converts Redmine[http://www.redmine.org]'s project searchbox with Chosen[http://harvesthq.github.com/chosen/] combobox.

= Features

- Chosen[http://harvesthq.github.com/chosen/] is a JavaScript plugin that makes long, unwieldy select boxes much more user-friendly

= Compatible Versions

- Redmine 2.1.x
